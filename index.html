AI Article Analyzer

In this project, you will build a program that analyses an article using pre-trained Transformer models. Your program will perform two tasks — summarise the article using a pre-trained summarisation model and then analyse the sentiment of the generated summary. This project demonstrates how multiple NLP tasks can be combined to analyse long text automatically.

Step 1: Create the Project Folder

  • Create a new folder called article_analyzer and open it in Visual Studio Code.

Step 2: Create the .env File

  • Create a new file in your project folder called .env and add the following line, replacing the placeholder with your actual Hugging Face API token:
    HF_TOKEN=your_token_here

Step 3: Load the Token

  • Create a new Python file called main.py in the same folder.
  • Import load_dotenv from the dotenv library and import Python's built-in os module.
  • Call the load_dotenv() function to load the variables from your .env file.
  • Use the getenv() function of the os module, passing 'HF_TOKEN' as the argument, to read the token and store it in a variable called HF_TOKEN.
  • Add a check to print Token loaded successfully if HF_TOKEN is not None, or Token not found if it is None.

Step 4: Create the Inference Client

  • Import InferenceClient from the huggingface_hub library.
  • Create a client object by calling InferenceClient() and passing token=HF_TOKEN as the parameter. Store it in a variable called client.

Step 5: Take Article Input

  • Print the following message: Paste your article below and press Enter twice when finished.
  • Create an empty list called lines.
  • Write a while True loop that reads one line of input at a time using input() and appends each line to lines. Inside the loop, write an if statement that checks whether the line is empty using line == ' ' and breaks out of the loop if it is.
  • Use ' '.join(lines) to combine all the lines into a single string and store it in a variable called text.

Step 6: Generate a Summary

  • Use the summarization() method of the client object, passing text as the first argument and model='facebook/bart-large-cnn' as the second argument. Store the result in result.
  • Access result['summary_text'] to extract the generated summary and store it in a variable called summary.
  • Print the summary.
